目录结构
========

## 知识点

* 讲解sails应用的目录结构

## 实战演习

* api:后台处理程序
  + controllers:控制器，映射路由URL
  + hooks:系统功能调用和改变
  + models:数据库实体模型
  + policies:访问策略
  + responses:定制http相应内容
  + services:定制全局服务类
* assets:网站静态文件根目录
* config:系统设置文件集
* tasks:打包工具
* views:数据模版布局
* app.js:应用程序启动入口
