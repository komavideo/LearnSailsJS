Sails.js开讲
============

## 概要

Sails.js是类似于Ruby on Rails的开发平台，采用Node.js进行开发，可以快速整合数据库资源，非常适合网站和API服务的构建。

## 类似框架

+ ExpressJS
+ Koa.js
+ Egg.js
+ etc.

## 官方网站

https://sailsjs.com/

## 使用版本

+ v0.12.14

## 基础知识

+ Javascript
+ HTML5/CSS
