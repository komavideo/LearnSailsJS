Sails.js的安装
=============

## 知识点

* npm方式安装

## 实战演习

### 官网

https://sailsjs.com/get-started

~~~bash
$ npm show sails
$ [sudo] npm install -g sails@0.12.14
$ sails -v
$ sails help
#建立一个应用程序
$ sails new myweb
$ cd myweb
$ ls -l
$ sails lift
~~~

浏览器确认启动页面。
